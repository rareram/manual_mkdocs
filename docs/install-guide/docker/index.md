# Docker 설치 가이드

Docker는 애플리케이션을 신속하게 구축, 테스트 및 배포할 수 있는 컨테이너(Container) 기술 기반의 오픈소스 플랫폼입니다. 애플리케이션과 그에 필요한 모든 종속성(라이브러리, 시스템 도구, 코드, 런타임 등)을 컨테이너라는 격리된 환경에 패키징하여, 어떤 환경에서든 동일하게 실행되도록 보장합니다.

이 기술은 "내 컴퓨터에서는 잘 됐는데, 서버에서는 왜 안 되지?" 와 같은 고질적인 문제를 해결하고, 개발, 테스트, 운영 환경의 일관성을 유지하는 데 핵심적인 역할을 합니다.

이 가이드에서는 다양한 운영체제에 Docker를 설치하는 방법을 안내합니다.

- **[Docker 설치](./docker-install.md)**: Windows, macOS, 그리고 다양한 Linux 배포판(Ubuntu/Debian, RHEL/CentOS) 환경에 맞는 Docker 설치 방법을 안내합니다.
