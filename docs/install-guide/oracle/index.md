# 개요

이 가이드는 Oracle 데이터베이스를 설치하고, `TableDiff` 기능 테스트에 필요한 사용자와 테이블을 생성하는 방법을 안내합니다.

설치는 다음 두 단계로 진행됩니다.

1.  **[Oracle DB 설치](./oracle-install.md)**: Docker 또는 Windows 네이티브 환경에 Oracle Database Express Edition (XE)를 설치합니다.
2.  **[데이터베이스 및 사용자 생성](./database-setup.md)**: 설치된 Oracle DB에 접속하여 테스트용 사용자를 만들고 샘플 테이블을 구성합니다.

각 문서를 순서대로 따라 진행하는 것을 권장합니다.
